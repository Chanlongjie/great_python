# 运算符

1. ### 算数运算符

   +

   -

   *

   /

   //（取整）

   %（取余）（0对任何数字取余都为0）

   **（幂）

   表达式：字母或数字组成（字母含有值）

   

2. ### 比较运算符

   ==

   !=

   (>)

   <

   <=

   (>=)

   

3. ### 逻辑运算符

   and	(布尔与	如果 x 为 False，x and y 返回 False，否则它返回 y 的计算值。)

   or	（布尔或	如果 x 是非 0，它返回 x 的值，否则它返回 y 的计算值。）

   not	（布尔非	如果 x 为 True，返回 False 。如果 x 为 False，它返回 True。）

   

4. ### 赋值运算符

   =	（a = b）

   +=	(a = a + b)

   -=	(a = a - b)

   *=	(a = a * b)

   /=	(a = a / b)

   //=	(a = a // b)

   %=	(a = a % b)

   **=	(a = a * * b)
